## What's new in 1.1.3

- **Input:** Windows Ink (pointer / mouse hook) is **preferred**; WinTab is **fallback** for older driver setups.
- **Logging:** Session log always written to `%APPDATA%\SketchASMR\sketch_asmr.log` (rewritten each launch). First pen contact logs whether **Windows Ink** or **WinTab** is active.
- **Settings UI:** Group **Loaded Sounds**; buttons **Load sounds**, **Load URL**, **Open Sounds**; version line at bottom of Settings.
- **Docs:** [Troubleshooting](https://github.com/JanitorPuppo/SketchASMR/blob/v1.1.3/README.md#troubleshooting) for Huion + Photoshop; sample [`PSUserConfig.txt`](https://github.com/JanitorPuppo/SketchASMR/blob/v1.1.3/PSUserConfig.txt) to force Photoshop to use WinTab when Windows Ink is disabled in the tablet driver.
